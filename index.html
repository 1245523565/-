目录切换

  • cd / 切换到根目录
  • cd /usr 切换到根目录下的usr目录
  • cd ../ 切换到上一级目录
  • ./ 为当前目录
  • cd - 切换到上次访问的目录

目录查看

  • ls查看当前目录下的所有目录和文件 (查不到隐藏的文件)
  • ls -a 查看当前目录下的所有目录和文件 (包括隐藏的文件)
  • ls -l 以列表形式查看当前目录下的所有目录和文件
  • ls /usr 查看usr下的所有目录和文件

目录的增删改查

创建目录

  • makdir 目录名 在当前目录下创建一个目录名的目录
  • makdir /usr/aaa 在usr目录下创建一个aaa的目录

删除目录

删除文件

  • rm 文件 删除当前目录下的文件
  • rm -f 文件 删除当前目录下的文件 不询问

删除目录

  • rm -r aaa 递归删除当前目录下的aaa目录
  • rm -rf aaa 递归删除当前目录下的aaa目录 不询问

全部删除

  • rm -rf * 删除当前目录下的所有文件和目录
  • rm -rf /* 删除根目录下的所有目录和文件

目录修改

重命名

  • mv 当前目录 新目录

  • mv aaa bbb 将当前目录下的aaa修改为bbb

    • mv可以对目录 文件 压缩包 进行重命名

剪切

  • mv 目录名称 目录的新位置
  • mv /usr/tmp/aaa /usr 将usr/tmp下的aaa目录剪切到usr目录下
  • mv可以对目录 文件 压缩包进行剪切

拷贝

  • cp -r 目录名称 目录拷贝的目标位置 -r表示递归
  • cp -r /usr/tmp/aaa /usr/tmp/bbb 将/usr/tmp下的aaa目录复制到/usr/tmp下的bbb目录内
  • cp可以对目录 文件 压缩包进行复制 对文件和压缩包进行操作是不用加递归 -r

查找

  • find 目录 参数 文件名称
  • find /usr/tmp -name 'a*' 查找/usr/tmp目录下所有以a开头的文件或目录

文件的增删改查

创建文件

  • touch 文件名
  • touch aa.txt 在当前目录创建一个名为aa的txt文件

删除文件

  • rm -rf 文件名
  • rm -rf /usr/tmp/aaa 删除/usr/tmp下的aaa目录

【vi编辑器的三种模式】

基本上vi分为三种状态，分为命令模式(command mode)、插入模式(Insert mode)和底行模式(last line mode)，各模式功能区分如下：

命令模式(command mode)

控制屏幕光标的移动，字符、字或行的删除，查找，移动复制某区段及进入插入模式(Insert mode)下，或者到底行模式(last line mode)。

  • 命令模式vi 下的常用名令

    • 控制光标的移动：K和↑、J和↓、H和←、L和→
    • 剪切当前行：dd
    • 查找：/字符
    • 进入编辑模式：i o a
    • 进入底行模式： :
    • 撤销操作：u
    • 恢复撤销的内容：ctrl+r
    • 将光标定位行首：0
    • 将光标定位行尾：$
    • 将光标定位到文件开头：gg
    • 将光标定位到文件末尾：G
    • 将光标定位到光标所在单词的起始处：b
    • 将光标定位到光标所在单词的结尾处：e
    • 将光标定位到光表所在单词的下一个单词的起始处：w
    • 保存并退出：shift+z+z
    • 删除光标所在字符：x
    • 粘贴到光标后：p
    • 粘贴到光标前：P 注意：以行为单位粘贴，会在光标的下一行开始粘贴
    • 拷贝光标所在行：y
    • 替换光标所在的字符：r
    • 进入替换模式：R
    • 将文件内所有的i替换为1： :%s/i/1/g

插入模式(Insert mode)

只有在插入模式下，才可以做文字输入，按ESC返回命令模式

底行模式(last line mode)

将文件保存或退出vi，也可以设置编辑环境，如寻找字符串，列出行号...等。

  • 底行模式下的常用命令

    • 退出编辑： :q
    • 强制退出： :q!
    • 保存并退出： :wq

打开文件

  • 注意：没有aa.txt会自动创建一个

编辑文件

使用vi编辑器打开文件后点击按键：i、a或者o进入编辑模式

  • i在光标所在字符前插入
  • I在光标所在行前插入
  • a在光标所在字符后插入
  • A在光标所在行后插入
  • o在光标所在行下面另起一新行插入
  • O在光标所在行上面另起一新行插入
  • s删除光标所在字符并进入插入模式
  • S清除光标所在行并进入插入模式

文件替换和查找

替换

语法:[addr]s/源字符串/目的字符串/[option]

[addr]表示检索范围,如:

"1,n":表示从第1行到n行 "%":表示整个文件,同"1,$" ".,$":表示从当前行到文件尾

[addr]省略时表示当前行

s:表示替换操作,其为substitute的缩写

[option] : 表示操作类型,如:

g:globe,表示全局替换

c:confirm,表示进行确认

p:表示替代结果逐行显示(Ctrl + L恢复屏幕)

i:ignore,不区分大小写

[option]省略时仅对每行第一个匹配串进行替换

如果在源字符串和目的字符串中出现特殊字符,如'/','<','>',','等需要前面加反斜杠\进行转义

常用命令示例:

#将当前行第一个a替换为b

:s/a/b/

#将当前行的所有a替换为b

:s/a/b/g

#将每行第一个a替换为b

:%s/a/b

#将整个文件的所有a替换为b

:%s/a/b/g

#将1至3行的第一个a替换为b

:1,3s/a/b/
